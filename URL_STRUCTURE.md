# URL Structure - E-Commerce Store

## SEO-Optimized URLs

All store URLs follow Miami-specific naming for SEO:

### Main Add-Ons Landing
- **URL:** `/Miami-Yacht-Charter-Add-Ons`
- **Purpose:** Main landing page with 4 category thumbnails
- **Navigation:** Listed in main menu as "ADD ONS"

### Store Pages

#### Catering Store
- **URL:** `/Miami-Yacht-Charter-Catering`
- **Features:** 
  - Category filtering (Sandwiches, Platters, Bowls, Vegetarian, Desserts)
  - 68 products total
  - Minimum 2 platter order
  - 36-hour advance notice required

#### Water Toys Store
- **URL:** `/Miami-Yacht-Charter-Water-Toys`
- **Features:**
  - 7 products (Seabob, Flitescooter, Water Sports Boat, etc.)
  - Max quantity limits (Seabob: 2, Flitescooter: 1)
  - Liability waivers required

#### Flowers Store
- **URL:** `/Miami-Yacht-Charter-Flowers`
- **Features:**
  - 11 floral arrangements
  - 3 size options per arrangement (Small/Medium/Large)
  - Price range: $89-$199

#### Bachelorette Packages Store
- **URL:** `/Miami-Yacht-Charter-Bachelorette-Packages`
- **Features:**
  - 4 package options
  - Price range: $199-$399
  - Optional drone video add-on

## URL Benefits

1. **SEO:** All URLs include "Miami Yacht Charter" keywords
2. **Readable:** Clear, descriptive paths
3. **Shareable:** Professional, branded URLs
4. **Consistent:** All follow same naming pattern

## Navigation Flow

```
Homepage
  └─ Main Menu → ADD ONS
      └─ /Miami-Yacht-Charter-Add-Ons
          ├─ Catering Thumbnail → /Miami-Yacht-Charter-Catering
          ├─ Water Toys Thumbnail → /Miami-Yacht-Charter-Water-Toys
          ├─ Flowers Thumbnail → /Miami-Yacht-Charter-Flowers
          └─ Bachelorette Thumbnail → /Miami-Yacht-Charter-Bachelorette-Packages
```

## Breadcrumb Pattern

All store pages include breadcrumb:
```
Add-Ons / [Store Name]
```

Example: `Add-Ons / Catering`

Links back to `/Miami-Yacht-Charter-Add-Ons`
